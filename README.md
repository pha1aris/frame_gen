# FPGA 帧生成与帧同步验证工程

本仓库用于保存 FPGA 帧生成、帧同步、锁定检测和仿真验证相关代码。工程内容包括 Vivado 工程文件、RTL 源码、仿真脚本以及用于帧同步算法调试的辅助文件。

## 目录说明

- `src/`：手写 RTL 源码，包含帧生成、帧同步和锁定检测相关模块。
- `sim/`：仿真脚本和仿真工作目录。建议只维护脚本、激励和必要说明，仿真生成的 `work/`、波形、transcript 等文件不建议提交。
- `frame_gen/`：Vivado 工程目录，包含工程配置、源文件引用和 IP 生成文件。

## Git 维护建议

推荐提交 `.v`、`.sv`、`.xdc`、`.tcl`、`.do`、`.md` 等源码和脚本文件。仿真数据库、临时文件、Vivado 缓存、日志和生成产物建议通过 `.gitignore` 排除。

## 当前状态

当前 GitHub 远端为 `pha1aris/frame_gen.git`，主分支为 `main`。
